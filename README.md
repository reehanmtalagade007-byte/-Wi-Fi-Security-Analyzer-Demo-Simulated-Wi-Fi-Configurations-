# 🔐 Wi-Fi Security Analyzer

A frontend demo tool that simulates Wi-Fi networks and analyzes their security level.

## Features

- Simulated Wi-Fi configurations
- Encryption type analysis
- Security score calculation
- Risk assessment
- Modern responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Disclaimer

This project does not scan real Wi-Fi networks.
It uses simulated data for educational purposes.
